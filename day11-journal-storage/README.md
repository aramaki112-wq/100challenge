# 100アプリチャレンジ DAY11

## アプリ名

永続化日記アプリ（localStorage版）

---

## 概要

DAY9の日記アプリを発展させ、ブラウザのlocalStorageを利用して日記を保存できるようにしました。

ブラウザを閉じても内容が保持され、再度開いたときに自動で復元されます。

---

## 機能

* 日記を追加
* 日記を削除
* localStorageへ保存
* 起動時に自動復元
* スマホ対応

---

## 使用技術

* HTML
* CSS
* JavaScript
* localStorage
* JSON.stringify()
* JSON.parse()

## 学習ポイント

今回のテーマは「データを保存すること」です。

画面を直接操作するのではなく、

データ（配列）

↓

保存

↓

画面描画

という設計を学びました。

この考え方は、今後作成するTodoアプリ・家計簿・将棋振り返りアプリなどにも応用できます。
