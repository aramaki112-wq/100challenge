# DAY13 文字数カウンターアプリ

## アプリ概要

入力した文章の

- 文字数
- 単語数
- 行数

をリアルタイムで表示するアプリです。

さらに、

- LocalStorageによる自動保存
- Markdown簡易プレビュー

にも対応しています。
