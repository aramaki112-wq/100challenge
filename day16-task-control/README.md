## 概要

DAY16では、DAY15で作成したタスク管理アプリを拡張し、より実用的なデータ構造へ発展させました。

今回のテーマは、

**「将来の機能追加を見据えた設計」**

です。

見た目の変化は少ないですが、開始日・期限日・完了状態を追加することで、今後予定しているカレンダー表示や検索機能、フィルター機能へ発展できる土台を作成しました。

# データ構造

今回のTodoは次のObjectで管理します。

```javascript
{
    task,
    priority,
    category,
    memo,
    startDate,
    dueDate,
    completed
}
```

今後はこのObjectを変更せずに、

* 検索
* 並び替え
* カレンダー表示
* 今日の予定
* 期限切れ一覧
* 統計表示

などを追加していきます。
