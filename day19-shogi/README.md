# DAY19 将棋棋譜再現アプリ Ver1

## 概要

DAY19では、100アプリチャレンジの休憩回として、将棋棋譜再現アプリ Ver1を制作しました。

本アプリの目的は、将棋のルールを実装することではありません。

「順番を持つデータ」と「現在位置(State)」を使った設計を学ぶことを目的としています。

---

## 学習テーマ

* 配列・オブジェクトによる棋譜データ管理
* Stateによる現在位置管理
* Event → State → Render の流れ
* LocalStorageによるデータ保存
* Renderの責務分離
* 境界チェック（最初・最後）
* 初期状態からの再構築という考え方

---

## 主な機能

* 9×9将棋盤表示
* 初期局面表示
* サンプル棋譜表示
* 現在手数表示
* 現在の指し手表示
* 棋譜一覧表示
* 現在手のハイライト
* 「進む」
* 「戻る」
* 「最初へ」
* 「最後へ」
* ボタンの自動無効化
* LocalStorage保存・読み込み

## データ構造

棋譜は一手ごとにObjectとして管理します。

```javascript
{
  notation: "▲７六歩",
  from: { row: 7, col: 7 },
  to: { row: 6, col: 7 },
  piece: "歩"
}
```

表示用データと処理用データを一つのまとまりとして扱うことで、拡張しやすい設計にしています。

---

## 設計のポイント

このアプリでは次の流れを中心に設計しています。

```text
Event
↓
State変更
↓
Render
```

さらに、

```text
初期状態
＋
現在位置までの履歴
＝
現在局面
```

という再構築方式を採用しています。

## 100アプリチャレンジ

100回設計し、100回改善し、100回考えることで、自分の力で設計・実装できるエンジニアになることを目指しています。
