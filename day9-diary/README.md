# DAY9 日記アプリ

## 概要

HTML・CSS・JavaScriptで作成した日記アプリです。

カレンダーから日付を選択し、
日記を入力・保存できます。

保存にはlocalStorageを使用しています。

---

## 学習内容

- Dateオブジェクト
- localStorage
- JSON.stringify()
- JSON.parse()
- オブジェクト管理
- カレンダー生成

---

## 機能

- 📅 カレンダー表示
- 📝 日記入力
- 💾 保存
- 🔄 読み込み
- ✏️ 上書き保存
- 🕒 最終更新日時表示
