# DAY10 World Clock Storage

## 概要

DAY8 世界時計アプリを発展させ、`localStorage` を利用して登録した都市をブラウザに保存できるようにしました。

## 主な機能

- 世界時計表示
- 都市追加
- 都市削除
- localStorageによる保存
- 次回起動時の自動復元
- スマホ最適化

## 学習内容

- localStorage
- JSON.stringify()
- JSON.parse()
- 配列の永続化
- データと画面を分ける設計

## Version

- DAY8 v1.0：世界時計
- DAY10 v2.0：保存・復元機能を追加（本アプリ）

## 次の目標

DAY11では、さらにデータを扱うアプリを作成し、保存・更新・削除の流れをより深く学習します。
