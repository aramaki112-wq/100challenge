# Bonus Challenge

Todoを

```text
[
  {
    text:"買い物",
    completed:false
  }
]
```

というオブジェクト配列で管理できるよう挑戦してみよう。

これは今後、

* 完了状態
* 日付
* 優先順位
* カテゴリー

などを追加するときの基本設計になります。
